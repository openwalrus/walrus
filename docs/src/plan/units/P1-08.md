# P1-08: Implement SqliteMemory CRUD + FTS

## Goal

Implement basic Memory trait methods (get, set, remove, entries, compile, store) on
SqliteMemory, backed by SQLite with FTS5.

## Depends On

P1-07.

## Design Decisions

- `get()` updates `accessed_at` and `access_count` on each access (for temporal decay).
- `set()` uses `INSERT OR REPLACE`. FTS triggers handle index updates.
- `entries()` returns all rows. `compile()` uses default impl (calls entries).
- `store()` handles metadata (JSON text) and embedding (BLOB).
