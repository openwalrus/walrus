# P1-05: Add Embedder Trait

## Goal

Define the Embedder trait in walrus-core: single async method, text to vector.

## Depends On

None.

## Design Decisions

- Lives in walrus-core (walrus-sqlite needs it for optional vector search).
- Returns `Vec<f32>` — no ndarray dependency.
- Takes `&self` for Arc-compatibility.
- Must be object-safe: `Box<dyn Embedder>` should compile.

## Files

- **CREATE** `crates/core/src/embedder.rs`
- **MODIFY** `crates/core/src/lib.rs` — add mod, re-export Embedder

## Tests

- `embedder_object_safe` — compile-time check that `Box<dyn Embedder>` works

## Done When

- [ ] `cargo check --workspace` passes
- [ ] Embedder re-exported from walrus-core
- [ ] `Box<dyn Embedder>` compiles
