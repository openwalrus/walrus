# crabtalk-telegram

Telegram bot gateway for [Crabtalk](https://github.com/crabtalk/crabtalk).

Connects Telegram chats to Crabtalk agents via the daemon's Unix socket. Supports
streaming replies, MarkdownV2 formatting, hub package management commands, and
per-user access control.

## Usage

```bash
crabtalk-telegram telegram start
```
